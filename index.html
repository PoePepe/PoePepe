    PoePepe

    Meet PoePepe, an easy-to-use Life Search manager for Path of Exile. It helps you search for items in real-time and make offers without leaving the game. Simplifying your trading experience!

    If you enjoy PoePepe. Support us on Patreon with just $3 a month. Your support keeps PoePepe strong!

    What can it do?

    • Manage multiple Life Searches in a single place.
    • Automatically send instant whispers, without losing time.
    • Receive notifications about new items and their details without the need to leave the game.

    How to start

    • Install and run the latest portable release.
    • Add your Session ID in the Settings tab.

    Demonstration

    Common Problems

    If you encounter an issue please report it.

    • Make sure pathofexile.com/trade is working and check if the provided link is valid.
    • Your Session ID might change. Keep it valid. Logging in and out refreshes your session ID.
    • You can only have 20 connections at once. It's a limit on pathofexile.com/trade, and you can't have more in your browser either.
